# RACI Matrix

## Purpose

This matrix defines management accountability for the documentation and delivery model around `JiraGami`.

`RACI` means:
- `Responsible`
- `Accountable`
- `Consulted`
- `Informed`

## Roles

- Project Manager
- Delivery Manager
- Product Lead
- Implementation Owner
- Quality Assurance (QA) Manager
- Stakeholders

## Management Responsibility Matrix

### Product definition

- Responsible: Product Lead
- Accountable: Project Manager
- Consulted: Delivery Manager, Quality Assurance (QA) Manager
- Informed: Stakeholders

### Implementation configuration

- Responsible: Implementation Owner
- Accountable: Delivery Manager
- Consulted: Product Lead, Project Manager
- Informed: Stakeholders

### Delivery operations

- Responsible: Delivery Manager
- Accountable: Project Manager
- Consulted: Quality Assurance (QA) Manager, Product Lead
- Informed: Stakeholders

### Quality Assurance (QA) routing

- Responsible: Quality Assurance (QA) Manager
- Accountable: Delivery Manager
- Consulted: Project Manager, Product Lead
- Informed: Stakeholders

### Documentation governance

- Responsible: Project Manager
- Accountable: Project Manager
- Consulted: Product Lead, Delivery Manager, Implementation Owner
- Informed: Stakeholders
